# Sales Data Analysis – Power BI Dashboard
# Project Overview

This project analyzes sales performance using Power BI to provide actionable business insights.
The dashboard helps in understanding revenue trends, customer behavior, and product performance for better decision-making.

# File Included

SalesDataAnalysis.pbix → Interactive Power BI dashboard file

#  Objectives

Analyze overall sales performance

Identify top-performing products and regions

Track revenue and profit trends

Understand customer buying behavior

Support business strategy and forecasting

#  Key Insights Covered

Total Sales & Revenue

Monthly & Yearly Sales Trends

Region-wise Sales Performance

Product Category Analysis

Customer Contribution

Profitability Overview

# Tools & Technologies

Microsoft Power BI

Excel / CSV (Data Source)

DAX (Data Analysis Expressions)

# Dashboard Features

Interactive filters (Region, Product, Time period)

Dynamic charts and tables

KPI cards for quick insights

Visual comparison of performance

# How to Use

Download the SalesDataAnalysis.pbix file

Open it in Microsoft Power BI Desktop

Click on slicers and visuals to explore insights

Use filters to analyze specific regions, products, or time periods

# Business Use Cases

Sales performance tracking

Revenue forecasting

Product demand analysis

Marketing strategy optimization

Regional sales comparison
